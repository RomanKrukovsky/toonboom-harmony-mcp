# Commercial Sales Offer: Harmony Autopilot MCP

## We automate Toon Boom Harmony production routines:
- **Scene setup**: Automatic resolution, frame rate, and workspace loading.
- **Asset import**: Importing backgrounds, character rigs, and overlays onto proper layers.
- **Character rig placement**: Positioning rigs using pre-defined coordinate templates.
- **Audio import**: Linking audio tracks to character layers.
- **Rough lipsync**: Automated mouth chart assignment based on audio waves.
- **Preview render**: Background rendering and delivery of low-resolution scene previews.
- **Scene audit**: Automatic network verification for broken nodes and empty exposure levels.
- **Production reports**: Time and cost-saving metrics sent directly to managers.

## Key Values for Animation Studios:

* **Less Manual Routine**: Animators focus on acting, movement, and quality rather than dragging files and renaming nodes.
* **Faster Scene Assembly**: AI Operator prepares ready-to-animate scenes in minutes.
* **Fewer Human Errors**: No missing rigs, incorrect frame rates, or broken composite connections.
* **Instant Client Review**: Automatic rendering of preview packages ready for the director's review.
* **Cheaper Production Costs**: Minimizes high-stress tasks for Technical Directors and reduces workflow cost.
