# Editable AI Animation

Unlike standard video generators that output flat MP4 files, this engine creates **editable Toon Boom Harmony projects**.

## Core Concept

- AI generates the structure, timing, and initial keyframes.
- Human artists retain full control to edit, tweak, and perfect the animation.
- Final output is a fully editable `.xstage` file.

## Flow

`Prompt -> AI Planning -> Harmony Assembly -> Editable Project`
