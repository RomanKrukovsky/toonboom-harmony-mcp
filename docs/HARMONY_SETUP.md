# Настройка автоматизации Harmony Server

Для активации всех сетевых возможностей управления базами данных Harmony Server должен быть настроен для приема удаленных запросов сценариев.

## 1. Запуск сервера скриптов

Вы можете запустить Control Center в режиме TCP-сервера скриптов на хост-машине базы данных следующим образом:

**Linux / macOS**:
```bash
export TOONBOOM_REMOTE_SCRIPT=1234
Controlcenter -script -tcpPort 1234
```

**Windows**:
```bat
SET TOONBOOM_REMOTE_SCRIPT=1234
Controlcenter.exe -script -tcpPort 1234
```

## 2. Открытие портов в брандмауэре

Убедитесь, что порт `1234` открыт в вашей локальной сети для возможности отправки Telnet-команд от MCP-сервера в Control Center.
- Стандартный порт управления БД — `5678`.
- Порт обмена сообщениями с клиентами — `5680`.
- Ограничивайте доступ к этим портам рамками доверенной сети вашей студии.

## 3. Пользователь `usabatch`

> [!WARNING]
> Пользователь `usabatch` является служебной административной учетной записью Toon Boom по умолчанию, необходимой для выполнения фоновых задач векторизации и рендеринга.
> - **Никогда не удаляйте и не переименовывайте usabatch**, так как это полностью нарушит работу очередей пакетного рендеринга.
