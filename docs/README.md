# Документация MCP-сервера Toon Boom Harmony

Добро пожаловать в документацию MCP-сервера (Model Context Protocol) для Toon Boom Harmony. Данный сервер предоставляет интеллектуальным агентам (например, Claude, ChatGPT) официальные инструменты автоматизации для управления базами данных Harmony Server и локальными сценами.

## Содержание документации

1. [Инструкция по установке](INSTALL.md)
   - Пошаговое руководство по развертыванию сервера и его зависимостей.
2. [Руководство по конфигурации](CONFIGURATION.md)
   - Описание переменных окружения, автоматического обнаружения путей и переопределения настроек.
3. [Правила безопасности](SECURITY.md)
   - Описание мер безопасности: песочница путей, симуляция выполнения (dry-run) и токены подтверждения опасных операций.
4. [Настройка автоматизации Harmony Server](HARMONY_SETUP.md)
   - Инструкции по запуску сервера сценариев Control Center и конфигурации сетевых портов.
5. [Справочник инструментов (Tools)](TOOLS.md)
   - Полный перечень и форматы параметров всех MCP-инструментов, зарегистрированных на сервере.
6. [Примеры интеграции](EXAMPLES.md)
   - Настройка Claude Desktop, Cursor IDE и примеры вызовов сценариев.
7. [Решение проблем (Troubleshooting)](TROUBLESHOOTING.md)
   - Чек-лист исправления типовых ошибок (таймауты, блокировка портов, проблемы импорта библиотек).
8. [Ограничения версий и совместимость](LIMITATIONS.md)
   - Текущие границы функциональности, особенности JS-движка Harmony и методыcapability-detection.
