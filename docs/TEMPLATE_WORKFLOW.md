# Production Template Workflow Guidelines

Использование готовых шаблонов — ключевой элемент ускорения анимационного производства. В Harmony Autopilot шаблоны делятся на следующие типы:

## 1. Шаблоны сцен (`.xstage`)
Хранятся в папке `examples/templates/` или на сервере Control Center.
* Содержат стандартные группы нод: `Top/Composite`, `Top/Display`, `Top/Write`, `Top/Camera`.
* Настроены под стандарты разрешения и частоты кадров студии.
* Рекомендуется иметь отдельные шаблоны под форматы:
  * Горизонтальный 1920x1080 (YouTube/ТВ).
  * Вертикальный 1080x1920 (Shorts/TikTok).

## 2. Риги персонажей (`.tpl`)
Риг персонажа — это папка-шаблон Toon Boom Harmony, содержащая иерархию костей (bones), пегов (pegs), рисунков и деформаторов.
* При импорте рига автопилот подключает корневой Composite рига к основному композиту сцены.
* Все слои ртов должны соответствовать стандартной разметке ртов (например, рты A, B, C, D, E, F, G, X) для автоматического чернового липсинга.

## 3. Пресеты камеры (`.json`)
Файлы конфигурации, содержащие координаты ключевых кадров для ноды `Camera_Peg`.
* Пресет `slow_push_in` плавно изменяет масштаб по оси Z.
* Пресет `camera_shake` добавляет случайные колебания на пег камеры при взрывах/ударах.

## 4. Пресеты эффектов (`.json`)
Шаблоны нод Toon Boom Harmony для применения визуальных эффектов (например, Glow, Blur, Cutter).
* Автопилот создает новый узел эффекта, подключает к нему указанную ноду-цель в качестве источника и выводит результат в Composite.
