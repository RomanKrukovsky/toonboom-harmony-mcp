# Инструкция по установке

Следуйте этим шагам для сборки и развертывания MCP-сервера Harmony.

## Системные требования

1. **Node.js**: Версия 18.0.0 или выше.
2. **TypeScript**: Установлен глобально или как devDependency проекта.
3. **Python 3.9+**: Для использования встроенного и внешнего Python API Harmony.
4. **SQLite3**: Системный пакет для создания базы данных локального трекера.
5. **Toon Boom Harmony Premium**: Версии 22 или 24, установленная на хосте.

## Первичная настройка

1. Перенесите кодовую базу в целевой рабочий каталог (например, `toonboom-harmony-mcp`).
2. Установите npm-пакеты:
   ```bash
   npm install
   ```

## Сборка проекта

Скомпилируйте исходные файлы TypeScript в JavaScript-код в папке `dist/`:
```bash
npm run build
```

## Тестирование работоспособности

Запустите модульные тесты для проверки безопасности и правильности генерации Qt-скриптов:
```bash
npm test
```

## Запуск MCP-сервера

Запуск сервера вручную через стандартный ввод/вывод (stdio):
```bash
npm run start
```
Сервер считывает входящие JSON-RPC 2.0 сообщения из stdin и выводит ответы в stdout.
