# Ограничения версий и совместимость

Учитывайте следующие технические ограничения при разработке пайплайнов с использованием данного MCP-сервера:

## 1. Поддержка стандартов JavaScript движком Control Center

- Скриптовый движок Control Center базируется на модуле Qt Script, реализующем стандарт ECMAScript 5 (ES5).
- Избегайте использования синтаксиса ES6+ (таких как `const`, `let`, стрелочные функции `() =>`, операторы развертывания `...`) в теле ваших кастомных Qt-скриптов. В противном случае движок выбросит синтаксическую ошибку.
- Все методы класса `QtScriptBuilder` генерируют ES5-совместимый код.

## 2. Однопоточность Python API в Harmony

- API-интерфейс Python в Toon Boom Harmony должен выполняться строго в главном графическом потоке приложения.
- Мост Python автоматически оборачивает вызовы в блокировки `harmony.thread_lock()` / `harmony.run_on_main()`, если они присутствуют. Не пытайтесь организовывать параллельные потоки для изменения графа нод сцены без использования официальных механизмов синхронизации Toon Boom.

## 3. Локальный SQLite трекер

- SQLite-база данных трекера задач разработана исключительно как легковесная надстройка для сохранения метаданных в отсутствие Toon Boom Producer.
- Она **не синхронизируется** автоматически со служебными базами данных Harmony Server. Все записи о задачах, кадрах и ресурсах должны создаваться явными вызовами инструментов трекера `harmony.workflow`.
