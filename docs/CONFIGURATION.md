# Руководство по конфигурации

Настройка MCP-сервера Harmony производится с помощью переменных окружения (или файла `.env` в корневой папке проекта).

## Справочник переменных конфигурации

| Имя переменной | Описание | Значение по умолчанию |
|---|---|---|
| `HARMONY_INSTALL` | Корневой каталог установки Toon Boom Harmony. | Определяется автоматически по ОС. |
| `HARMONY_CC_BIN` | Полный путь к исполняемому файлу Control Center. | Определяется автоматически во внутренней bin. |
| `HARMONY_BIN` | Полный путь к исполняемому файлу HarmonyPremium. | Определяется автоматически во внутренней bin. |
| `HARMONY_PYTHON_PACKAGES` | Путь к директории \`python-packages\` Harmony. | Определяется автоматически во внутренней lib. |
| `HARMONY_CC_HOST` | Хост сервера скриптов Control Center. | `127.0.0.1` |
| `HARMONY_CC_PORT` | TCP-порт сервера скриптов Control Center. | `1234` |
| `HARMONY_CC_USER` | Имя пользователя базы данных для скриптов. | `usabatch` |
| `HARMONY_SCRIPT_TIMEOUT_MS`| Время таймаута для подпроцессов и Telnet. | `10000` (10 секунд) |
| `HARMONY_DRY_RUN_DEFAULT` | Включает симуляцию (dry-run) по умолчанию. | `true` (Рекомендуется для безопасности) |
| `HARMONY_ALLOW_DESTRUCTIVE` | Разрешает опасные/деструктивные действия. | `false` |
| `HARMONY_ALLOW_RAW_SCRIPTS` | Разрешает запуск сырых Qt-скриптов. | `false` |
| `HARMONY_ALLOWED_ROOTS` | Разрешенные папки для файловых операций. | Каталог проекта, `/tmp` |
| `HARMONY_LOG_DIR` | Папка для хранения структурированных логов. | `./logs` |

## Автоматическое обнаружение путей (Path Auto-Detection)

При запуске сервер сканирует стандартные папки операционной системы:
- **macOS**: Ищет каталоги внутри `/Applications/Toon Boom Harmony * Premium/` и извлекает бинарные папки из структуры `.app`.
- **Windows**: Сканирует каталог `C:\Program Files\Toon Boom Animation\Toon Boom Harmony * Premium` и нацеливается на `win64/bin`.
- **Linux**: Ищет исполняемые файлы в `/usr/local/ToonBoomAnimation/harmony_*`.

Если на вашей студии используются нестандартные пути, задайте значения переменных `HARMONY_INSTALL` или конкретные пути к бинарным файлам в файле `.env` для переопределения автоматических настроек.
