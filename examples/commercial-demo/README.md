# Демонстрационный режим сборки сцены (Commercial Demo)

Этот демонстрационный пакет позволяет запустить и визуализировать работу **Harmony Autopilot MCP** в полностью контролируемом режиме (симуляции или на реальном проекте).

## Что делает агент
Агент считывает файл `scene_plan.json`, строит пошаговый граф действий и выполняет их по очереди:
1. Запускает Toon Boom Harmony.
2. Создает сцену из шаблона производства с нужным разрешением и FPS.
3. Импортирует фон `lab.png` на отдельный слой таймлайна.
4. Импортирует риг персонажа `Scientist.tpl` и позиционирует его слева.
5. Импортирует аудио дорожку `sc_001.wav`.
6. Применяет черновой липсинк для персонажа по звуковой волне.
7. Добавляет медленный наезд камеры (`slow_push_in`).
8. Накладывает спецэффект свечения (`Glow FX`) на портал.
9. Сохраняет сцену и запускает рендеринг видео-превью.
10. Проводит аудит сцены на отсутствие битых слоев и выдает детальный отчет.

## Как запустить демо
Для запуска демо вызовите метод `harmony.autopilot.run_scene_plan` с параметрами:
```json
{
  "scenePlanPath": "examples/commercial-demo/scene_plan.json",
  "dryRun": false
}
```

Вы также можете сгенерировать коммерческий отчет об экономии времени с помощью:
```json
harmony.production.generate_time_savings_report(scenesCount: 10, hourlyRate: 35)
```

## Что считается успехом
* В логах выполнения (`harmony.autopilot.get_execution_log`) все шаги имеют статус `passed`.
* Отсутствуют ошибки сборки (broken connections).
* Создан превью-файл рендеринга.
* Время сборки составило менее 5 секунд в режиме симуляции (против ~3.5 часов ручной работы).

## Как записать видео-демо
1. Включите запись экрана.
2. Запустите автопилот.
3. Продемонстрируйте автоматическое появление слоев на таймлайне и рендеринг.
4. Покажите отчет об экономии ресурсов и времени (`generate_time_savings_report`) — это главный аргумент для продюсеров студий при продаже.
