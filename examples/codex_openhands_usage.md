# Инструкция по использованию с OpenHands / Codex

OpenHands и Codex могут использовать MCP-сервер Harmony путем настройки стандартного канала ввода/вывода (stdio).

## Конфигурация подключения

Добавьте следующую запись конфигурации в манифест инструментов MCP вашего агента:

```json
{
  "mcpServers": {
    "toonboom-harmony": {
      "command": "node",
      "args": ["/Users/romanmolodyko/Documents/toon-boom-harmony-mcp/dist/index.js"],
      "env": {
        "HARMONY_DRY_RUN_DEFAULT": "true",
        "HARMONY_ALLOW_DESTRUCTIVE": "false"
      }
    }
  }
}
```

## Инструкции для системного промпта агента

Рекомендуется добавить следующий текст в системные инструкции агента:
> "Вам доступны инструменты автоматизации Toon Boom Harmony. Используйте функции пространства имен `harmony.cc` для конфигурирования баз данных и создания окружений/проектов/сцен на сервере, и `harmony.scene` для управления локальными нодами и свойствами открытых файлов проекта."
